Privacy Policy

This Privacy Policy explains how the Sudoku Game Chrome Extension (hereinafter referred to as "we" or "the Extension") collects, uses, and protects your information. We value your privacy and are committed to protecting your personal information.

Information Collection

The Extension collects very limited information, which includes only:

  • Language Preference: Your chosen interface language (Chinese or English)

Information Usage

We use the collected information solely for:

  • Saving your language preference to display the appropriate interface language on your next visit
  • Syncing your language settings across different devices through Chrome sync functionality

Data Storage

All data is stored using the Chrome Storage API, specifically:

  • Data is stored locally in your Chrome browser
  • Data may sync across your devices if you're signed into Chrome
  • We do not upload any data to external servers

Permissions

The Extension requests the following permissions:

  • storage: Used to save language preference settings
  • downloads: Used to save Sudoku puzzles as image files

Data Security

We protect your data through the following measures:

  • Only collecting necessary configuration information
  • Using Chrome's official secure APIs
  • Not collecting any personal identification information
  • Not sharing any data with third parties

User Rights

As a user, you have the following rights:

  • Change language settings at any time
  • Delete stored data through Chrome's extension management features
  • Automatic deletion of all related data upon extension uninstallation

Privacy Policy Updates

We may update this Privacy Policy from time to time. When we make significant changes, we will notify you through extension updates. Continued use of the Extension indicates your acceptance of the latest Privacy Policy.
