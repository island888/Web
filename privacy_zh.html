隐私政策

本隐私政策说明了数独游戏 Chrome 扩展程序（以下简称"我们"或"本扩展"）如何收集、使用和保护您的信息。我们重视您的隐私，并致力于保护您的个人信息。

信息收集

本扩展程序收集的信息极其有限，仅包括：

  • 语言偏好设置：记录您选择的界面语言（中文或英文）

信息使用

我们收集的信息仅用于：

  • 保存您的语言偏好，以便在您下次使用时显示合适的语言界面
  • 通过 Chrome 同步功能在您的不同设备间同步语言设置

数据存储

所有数据都使用 Chrome Storage API 进行存储，具体包括：

  • 数据存储在您的 Chrome 浏览器本地
  • 通过 Chrome 同步功能，数据可能会同步到您登录的其他设备
  • 我们不会将任何数据上传到外部服务器

权限说明

本扩展程序请求的权限包括：

  • storage：用于保存语言偏好设置
  • downloads：用于将数独题目保存为图片文件

数据安全

我们采取以下措施保护您的数据：

  • 仅收集必要的配置信息
  • 使用 Chrome 官方提供的安全 API
  • 不收集任何个人身份信息
  • 不与第三方共享任何数据

用户权利

作为用户，您拥有以下权利：

  • 随时更改语言设置
  • 通过 Chrome 浏览器的扩展管理功能删除存储的数据
  • 卸载扩展程序时自动删除所有相关数据

隐私政策更新

我们可能会不时更新本隐私政策。当我们进行重大更改时，将通过扩展程序更新通知您。继续使用本扩展则表示您同意最新的隐私政策。
